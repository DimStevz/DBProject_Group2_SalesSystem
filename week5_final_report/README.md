# InventoryDB

A database inventory system with a minimalistic UI to serve product management and inventory tracking, aimed towards small businesses.

## Usage

1. Install dependencies by `python3 -m pip install -r requirements.txt`.
2. Run instance by `flask run` in the folder.
3. Open the local instance on `http://127.0.0.1:5000/`.
4. By default, the user with the username `admin` and password `admin` is made. Make sure to change it following the setup!

## Tools

This project relies on Flask and SQLite for its backend. Its minimalism is helped by the writing of the frontend using pure HTML+CSS+JS without any bloaty frameworks.
